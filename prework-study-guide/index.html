HTML

  • The head element contains info about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content isnide an element
  • img selector assigns property values to any image element in the markup

Git

  • git status: checks what branch you are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout: switches branch
  • git add:changes from branch are placed in staging
  • git -A: adds all changes
  • git -m"":leave a message
  • git pull: Updates from branch
  • origin: source for pull
  • git push: Updates to branch

JavaScript

  • Varible is a container that allows us to store data in code
  • Control flow is the order in which a computer executes code
  • Functions are a piece of code condensed to be reuseable
  • For loops continually executes code usually based on if and if else statements
  • Array is a varible with multiple kinds of data stored in it
  • To find a specific piece of data in a array you need to keep track of the varible index
